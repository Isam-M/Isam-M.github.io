Hvordan automatisk kjøre programmer når filer endres

    rg --files | grep -v index.html | entr -rc node ./build.js

    Krever:
        ripgrep (brew install ripgrep)
        entr (brew install entr)

    Hvis du skal kjøre kommandoen i ditt eget script, må du gjøre scriptet kjørbart.
    Det kan du gjøre med chmod +x.

    For eksempel:

        chmod +x ./mittscript
